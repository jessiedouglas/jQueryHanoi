jQueryHanoi
===========

This is a simple Towers of Hanoi app using JavaScript/jQuery.

#### Running this App
1. `git clone` or download this repository.
2. Open 'index.html' in the browser (you can drag and drop the file!)
3. Play the game!

### Playing the Game
1. You have to refresh to start over :(
